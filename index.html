Welcome to Ajvm Pages.

Ajvm is a hobby java virtual machine, which i just want to know how a jvm works.

from now on, the ajvm can only do two tings:

1、 java bytecode diassember. 2、 execute several bytecodes: iload_n, istore_n, aload_n, astore_n, iadd, isub, bipush, invokespecail, invokestatic, invokevirtual, goto, return, ireturn, if_icmpge, putfiled, new, dup.
